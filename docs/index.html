Plugram

Custom Telegram client with Liquid Glass UI, plugin system in .plugram language, and 30+ built-in plugins.

📦 Download 🔌 Plugins 📖 Docs

Features

🎨
Liquid Glass UI
Glassmorphism design with fluid animations and dark theme. PWA — install on your phone as a native app.
🧩
Plugin System
Write plugins in .plugram — a custom scripting language. Hook into messages, commands, and events.
👻
Ghost Mode
Read messages without being seen. Disable read receipts and stay invisible while reading chats.
🎤
Voice Messages
Record and send voice messages directly from your browser. Built-in microphone recorder.
👥
Multi Account
Switch between multiple Telegram accounts. Each session stored separately with full encryption.
🎯
Ghost Commands
Typing indicators, online statuses, sent/read confirmation, pinned messages, and full message search.
🎭
Custom Themes
Dark/Light themes with custom accent colors. Design your own Liquid Glass theme with CSS variables.
📦
30+ Plugins
Auto-reply, spam guard, word filter, reminders, polls, keyword alerts, translator, and many more — all in .plugram.
🔗
Plugin Store
Browse and install community plugins from bitplugg/plugram-plugin directly from the settings panel.

Installation

1. Clone the repo

git clone https://github.com/bitplugg/plugram.git
cd plugram

2. Install & run server

cd server && npm install && npm start

3. Install & run client

cd client && npm install && npm start

Open http://localhost:3000 and enter your Telegram API credentials from my.telegram.org

Plugins: Clone bitplugg/plugram-plugin into the plugins/ folder, or browse the store from Settings.
