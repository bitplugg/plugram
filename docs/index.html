Features

🎨
Liquid Glass UI
Glassmorphism design with fluid animations and dark theme. PWA — install on your phone as a native app.
🧩
Plugin System
Write plugins in .plugram — a custom scripting language. Hook into messages, commands, and events.
👻
Ghost Mode
Read messages without being seen. Disable read receipts and stay invisible while reading chats.
🎤
Voice Messages
Record and send voice messages directly from your browser. Built-in microphone recorder.
👥
Multi Account
Switch between multiple Telegram accounts. Each session stored separately with full encryption.
📊
Polls & Voting
Create polls with multiple options, vote in real-time, and see percentage results.
📍
Location Sharing
Share your current location with one tap. Opens directly in Google Maps.
🔇
Mute / Archive / Block
Full chat management — mute notifications, archive conversations, block users.
📱
Mobile First
Swipe gestures, bottom navigation bar, pull to refresh. Works great on any screen size.
🔌
Auto-Reconnect
Persistent WebSocket connection with exponential backoff. Never miss a message.
🎥
Video Player
Full video controls: play/pause, seek, volume, speed control (0.5x-2x), Picture-in-Picture.
🎨
Custom Themes
Dark/Light themes, adjustable font size (12-20px), three density modes, toggle animations.
📎
Drag & Drop Files
Drop images, videos, documents directly into the chat. Preview before sending.
🔒
Saved Messages
Quick access to your Saved Messages chat from sidebar and bottom nav.
🎭
Customization
Font size, density (compact/normal/spacious), and toggle animations on/off.
📦
30+ Plugins
Auto-reply, spam guard, word filter, reminders, polls, keyword alerts, translator, and many more.

❓ FAQ

Is this a bot?
No. Plugram uses your personal Telegram account via MTProto (gramJS). It's a full client, not a bot.
Is my data safe?
Yes. Everything runs locally on your own server. No third-party servers or cloud services are involved.
Can I use it on mobile?
Yes. It's PWA-enabled — open in Chrome or Safari and "Add to Home Screen" for a native-like experience with bottom navigation and swipe gestures.
How do plugins work?
Plugins are written in the .plugram scripting language (Tcl-like syntax) and run server-side. They hook into message events, commands, and system events.
Do I need an API ID?
Yes. Get one free at my.telegram.org/apps. Enter it in the Settings panel on first launch.
Can I contribute plugins?
Absolutely! Fork bitplugg/plugram-plugin, write your .plugram file, and open a pull request.

Installation

1. Clone & enter

git clone https://github.com/bitplugg/plugram.git
cd plugram

2. Install & run server

cd server && npm install && npm start

3. Install & run client

cd client && npm install && npm start

Open http://localhost:3000 and enter your Telegram API credentials.

Plugins: Clone bitplugg/plugram-plugin into plugins/, or install from the store in Settings.
